Prérequis — Avant de commencer
Les deux machines
1
Vérifier que la transaction est nécessaire
Ne jamais signer sous la pression ou dans l'urgence. Confirmer l'adresse destination, le montant et les frais.
Tu ne hodl pas pour rien — chaque sortie de cold wallet est irréversible.
2
Préparer le matériel
USB Clé Tails bootable
USB Clé USB vierge ou propre pour transfert fichier PSBT (si pas QR)
PC guidesk connecté à Internet (pour broadcaster)
Tails PC air-gappé — guidesk en mode Tails (déconnecté réseau)
Si tu utilises guidesk pour les deux rôles : boot Tails d'abord (déconnecté), signe, reboot Linux normal, broadcaste.
3
Récupérer la seed / clé privée
Ouvrir le conteneur VeraCrypt → KeePass → récupérer la seed phrase (ou clé privée WIF) du wallet.
⚠ Ne jamais saisir la seed sur un PC connecté à Internet. Uniquement sous Tails, réseau coupé.
📡
Machine en ligne — Créer la transaction non signée (PSBT)
guidesk · Linux normal · connecté
1
Ouvrir Electrum en mode watch-only
Lancer Electrum sur guidesk (Linux normal, connecté). Le wallet watch-only contient la clé publique uniquement — aucune clé privée présente.
electrum
Si pas encore configuré : File → New/Restore → Standard wallet → Use a master key → coller la xpub.
2
Créer la transaction
Onglet Send → saisir l'adresse destination → montant → ajuster les frais (fee).
Cliquer Pay puis Advanced → vérifier les inputs/outputs.
Vérifier 2× l'adresse destination. Copier/coller, ne jamais retaper manuellement.
3
Exporter le PSBT (transaction non signée)
Dans la fenêtre de transaction : Export → choisir le format de transfert :
Sauvegarder le fichier .psbt sur la clé USB de transfert.
~/Bureau/transaction_unsigned.psbt
Utiliser une clé USB dédiée au transfert, pas la clé Tails.
Cliquer Show as QR — un QR code s'affiche à l'écran.
La machine Tails le scannera avec la caméra ou une app QR.
Plus propre que l'USB — aucun support physique partagé entre les machines.
🔐
Tails OS — Signer la transaction (air-gappé)
Tails · réseau COUPÉ
1
Booter sur Tails — Couper le réseau
Redémarrer guidesk sur la clé Tails (F11 au démarrage → sélectionner USB).
Au démarrage Tails : ne pas configurer le réseau, cliquer + (More options) → désactiver Tor/réseau.
⚠ CRITIQUE — Vérifier que le WiFi et l'Ethernet sont bien déconnectés avant de saisir quoi que ce soit.
ip link show
Tous les états doivent être DOWN ou NO-CARRIER.
2
Lancer Electrum sur Tails
Tails inclut Electrum. Le lancer en mode offline :
electrum --offline
Si premier lancement : File → New/Restore → Standard wallet → I already have a seed → saisir la seed phrase.
Le wallet créé sur Tails disparaît au reboot (Tails est amnésique). C'est voulu.
3
Importer le PSBT
Brancher la clé USB de transfert → Tools → Load transaction → From file → sélectionner le .psbt.
Tools → Load transaction → From QR code → scanner le QR affiché sur l'autre écran (ou photo sur le même écran).
4
Vérifier la transaction AVANT de signer
Dans la fenêtre de transaction vérifier impérativement :

Adresse destination — correspondre exactement à ce qui était prévu
Montant — vérifier en BTC ET en valeur fiat approximative
Frais (fee) — raisonnables par rapport au réseau actuel
Change address — doit être une adresse de ton wallet
⚠ Si QUOI QUE CE SOIT ne correspond pas — annuler immédiatement. Une transaction signée et broadcastée est IRRÉVERSIBLE.
5
Signer la transaction
Cliquer Sign → saisir le mot de passe du wallet Electrum (pas la seed, le mdp du wallet).
La transaction passe de unsigned à signed.
6
Exporter la transaction signée
Export → sauvegarder transaction_signed.psbt sur la clé USB de transfert.
Show as QR → la machine en ligne scannera ce QR pour récupérer la transaction signée.
Ne pas broadcaster depuis Tails — toujours depuis la machine en ligne.
📡
Machine en ligne — Broadcaster la transaction
guidesk · Linux normal · connecté
1
Rebooter sur Linux normal
Éteindre Tails → retirer la clé Tails → démarrer normalement sur Linux Mint.
Rebrancher Internet.
2
Importer la transaction signée dans Electrum
Ouvrir Electrum → Tools → Load transaction → From file → sélectionner transaction_signed.psbt sur la clé USB.
Ouvrir Electrum → Tools → Load transaction → From QR code → scanner le QR affiché par Tails.
3
Dernière vérification avant broadcast
Vérifier une dernière fois l'adresse destination et le montant dans la fenêtre Electrum.
Vérifier le statut : doit afficher Signed.
⚠ Dernier point de non-retour. Après broadcast : irréversible.
4
Broadcaster
Cliquer Broadcast dans Electrum.
Ou via le site mempool.space si Electrum pose problème :
https://mempool.space/fr/tx/push
Copier le hex de la transaction brute (Tools → Export → As hex) et le coller sur mempool.
5
Confirmer et suivre
Récupérer le TXID affiché par Electrum après broadcast.
Suivre la confirmation sur mempool.space :
https://mempool.space/fr/tx/TXID
Attendre 1 confirmation minimum avant de considérer la transaction comme valide.
✓ Transaction broadcastée — noter le TXID dans KeePass pour traçabilité.
✅
Récapitulatif — Transaction complète
Bilan
Checklist finale
Sécurité post-transaction
Supprimer le fichier .psbt de la clé USB
Fermer KeePass et démonter VeraCrypt
Éjecter la clé Tails et la ranger
Noter le TXID dans KeePass
Attendre 6 confirmations pour être serein
✓ Workflow air-gapped respecté — la clé privée n'a jamais touché une machine connectée à Internet.
